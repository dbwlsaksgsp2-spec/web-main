- **Google Calendar API 활용 일정 관리 및 데이터 추출**: `google-workspace` MCP 도구를 통해 캘린더에 신규 일정을 등록하고(생성), 특정 키워드('캠페인 기획 회의')를 포함하는 일정을 기간별로 필터링/조회하여 회의록을 자동 요약하는 흐름을 체득했습니다.
- **구글 드라이브/문서 통합 자동화 (Agent Execution)**: TOML 파일에 정의된 에이전트의 구조(도구 사용 권한, task 정의, outcome 목적지)에 맞추어 여러 클라우드 리소스(Sheet, Gmail, Calendar, Drive, Doc)를 결합하여 하나의 통합 문서 보고서를 빌드하고 자동으로 특정 폴더로 이동시키는 엔드 투 엔드 워크플로우를 완수했습니다.
- **로컬 CSV 파일의 구글 스프레드시트 마이그레이션**: 로컬에 분산된 CSV 텍스트 정형 데이터를 구글 API를 활용해 클라우드 스프레드시트의 특정 탭/셀 범위에 정확하게 매핑 및 마이그레이션하는 기법을 실습했습니다.
- **웹 검색 연동 가격 분석 자동화 및 데이터 쓰기**: TOML 에이전트의 정의를 바탕으로 스프레드시트의 비품 신청 데이터를 파악한 후, 실시간 웹 검색 엔진을 결합하여 최저가 순위와 배송 조건을 추출하고, 이를 다른 탭에 저장하며 결과를 Gmail 알림으로 피드백하는 전 과정을 자동화했습니다.
- **웹 뉴스 크롤링/파싱 및 지식 모델링 자동화**: 웹 검색 도구를 통해 실시간 데이터를 대량으로 피딩하고, 중복 노출도를 기준으로 중요도(임계값 5회 이상)를 측정하여 단일 보도자료 형태로 지식을 합성하는 LLM 에이전트 워크플로우를 구현했습니다.
- **클라우드 데이터 추출 및 감성 텍스트 알림장 자동 생성**: 구글 스프레드시트에 기입된 아이들의 일상 키워드(식사, 놀이, 낮잠 등) 데이터를 가공하여, 학부모 맞춤형 감성적 알림장 본문을 생성하고 대량 발송하는 메일 발송 워크플로우를 완수했습니다.
- **슬라이드 생성 및 객체 데이터 맵핑 자동화**: 구글 프레젠테이션 API의 batch_update 기능을 활용하여, 텍스트 상자 및 레이아웃을 정의하고 에이전트 작동 시나리오와 아키텍처 기반의 완성도 높은 발표자료(PPT)를 자동으로 빌드하는 과정을 학습했습니다.
- **자산운용가 의사결정 시뮬레이션 및 데이터 구조화**: 실시간 웹 검색 엔진을 통해 수집한 최신 글로벌 매크로 경제 동향을 자산운용 전문가 관점에서 분석하고, 정형화된 스프레드시트 양식으로 변환하여 클라이언트 메일로 피드백하는 금융 자동화 시나리오를 완수했습니다.
