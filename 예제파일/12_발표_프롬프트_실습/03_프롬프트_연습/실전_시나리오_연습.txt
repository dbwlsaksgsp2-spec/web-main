※ 한꺼번에 시키지 말고, 단계별로 확인하며 진행하세요!

═══════════════════════════════════════════════
📋 시나리오 A: 월간 매출 분석 → 보고서 → 발표
═══════════════════════════════════════════════

배경: 팀장님이 "이번 달 매출 정리해서 발표까지 해" 라고 했습니다.
파일: 02_발표_재료/월별매출.csv

【1단계】 데이터 파악
🗣️ "월별매출.csv를 열어 어떤 항목이 있고 총 몇 건인지 요약해줘."
→ 확인: 열 이름, 행 수, 데이터 범위를 파악했는가?

【2단계】 집계
🗣️ "제품별 총매출·월평균·최고/최저 월을 표로 정리해줘. 새 파일로 저장."
→ 확인: 숫자가 합리적인가? 원본은 그대로인가?

【3단계】 차트
🗣️ "월별 총매출 꺾은선 차트와 제품별 비중 원그래프를 엑셀에 넣어줘."
→ 확인: 차트가 데이터와 맞는가? 읽히는 크기인가?

【4단계】 보고서
🗣️ "집계와 차트를 포함한 '상반기 매출 보고서' PDF를 만들어줘. 표지·3줄 요약 포함."
→ 확인: 표지에 제목·날짜·작성자가 있는가?

【5단계】 PPT
🗣️ "보고서를 바탕으로 발표 슬라이드 8장을 만들어줘. 표지·목차·본문·요약·마무리 구조."
→ 확인: 한 장에 한 메시지인가?

【6단계】 대본
🗣️ "각 슬라이드에 30초 발표 대본을 만들어줘. 숫자 포함, 존대 말투."
→ 확인: 대본이 자연스럽게 읽히는가?

═══════════════════════════════════════════════
📋 시나리오 B: 고객 만족도 → 개선 보고 → 팀 발표
═══════════════════════════════════════════════

배경: "고객 피드백 정리해서 개선 방안 발표해" 라는 지시를 받았습니다.
파일: 01_보고서_원본/고객만족도_조사결과.txt, 02_발표_재료/고객만족도_월별추이.csv

【1단계】 데이터 요약
🗣️ "고객만족도_조사결과.txt 내용을 읽어 핵심 수치를 표로 정리해줘."
→ 확인: 종합 만족도, 항목별 점수, 등급별 차이가 정리됐는가?

【2단계】 추이 분석
🗣️ "고객만족도_월별추이.csv에서 카테고리별 월별 만족도 변화 꺾은선 차트를 만들어줘."
→ 확인: 상승/하락 트렌드가 보이는가?

【3단계】 비교 분석
🗣️ "등급별(베이직·프리미엄·엔터프라이즈) 만족도와 추천의향을 비교 막대 차트로 만들어줘."
→ 확인: 등급 간 차이가 시각적으로 드러나는가?

【4단계】 개선 보고서
🗣️ "분석 결과를 바탕으로 '고객 만족도 개선 보고서'를 만들어줘. 현황·원인·개선안 구조, 3줄 요약 포함."
→ 확인: 개선안이 구체적이고 실현 가능한가?

【5단계】 발표 자료
🗣️ "개선 보고서를 10장짜리 발표 슬라이드로 만들어줘. 차트 포함, 각 장 대본도."
→ 확인: 문제→원인→해결 흐름이 논리적인가?

═══════════════════════════════════════════════
📋 시나리오 C: 교육 결과 → 성과 보고 → 경영진 발표
═══════════════════════════════════════════════

배경: "이번 교육 결과 정리해서 경영진 보고해" 라는 지시입니다.
파일: 01_보고서_원본/신입사원_교육결과_보고.txt, 02_발표_재료/교육과정별_실적.csv

【1단계】 데이터 확인
🗣️ "교육과정별_실적.csv를 요약해줘. 과정이 몇 개고, 전체 평균 점수와 수료율을 알려줘."

【2단계】 분석
🗣️ "과정별 수료율과 만족도를 비교하는 이중 막대 차트를 만들어줘."
🗣️ "평균 점수가 가장 낮은 과정과 가장 높은 과정을 강조해줘."

【3단계】 보고서
🗣️ "교육 결과 보고서를 만들어줘. 과정별 성과표 + 개선 필요 과정 3개 + 3줄 요약. PDF 저장."

【4단계】 발표
🗣️ "경영진 10분 발표용 슬라이드를 만들어줘. 핵심 숫자를 크게, 차트 중심, 대본 포함."

【5단계】 패키지
🗣️ "보고서 PDF + PPT + 대본을 '교육성과_보고패키지' 폴더에 모아 zip으로 압축해줘."

📋 시나리오 D: 사업 계획 발표 준비
═══════════════════════════════════════════════

배경: "하반기 사업 계획을 이사회에서 발표해야 해요."
파일: 01_보고서_원본/하반기_사업계획_요약.txt, 02_발표_재료/분기별_손익요약.csv

【1단계】 내용 파악
🗣️ "하반기_사업계획_요약.txt 내용을 읽어 핵심 전략 4개와 예산을 표로 정리해줘."

【2단계】 재무 차트
🗣️ "분기별_손익요약.csv에서 매출·비용·이익 추이를 꺾은선 차트로 만들어줘."

【3단계】 PPT 제작
🗣️ "이사회 발표용 12장 슬라이드를 만들어줘. 전략별 1~2장씩, 재무 차트, 로드맵 슬라이드 포함."

【4단계】 대본 + 예상질문
🗣️ "슬라이드별 45초 대본을 만들어줘. 추가로 예상 질문 5개와 답변도 준비해줘."

【5단계】 핸드아웃
🗣️ "슬라이드 내용을 1장 요약 핸드아웃으로도 만들어줘. 이사님들이 회의 중 참고할 수 있게."
